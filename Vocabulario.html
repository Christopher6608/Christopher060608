Framework: es una estructura o conjunto de herramientas que proporcionan una base para desarrollar aplicaciones o proyectos. Es una plantilla o esquema que facilita el trabajo, permitiendo a los desarrolladores enfocarse en la lógica específica de la aplicación en lugar de reinventar la rueda. Los frameworks simplifican tareas, agilizan procesos y ayudan a lograr mayor productividad.

Aplicación: es un programa de software diseñado para realizar una tarea específica, como enviar correos electrónicos, jugar juegos, o gestionar finanzas. Se pueden encontrar en diferentes dispositivos, desde computadoras hasta teléfonos móviles.

Usuarios: es una persona que usa, o es destinatario de, un producto, servicio, o sistema, ya sea de forma individual o como parte de un grupo. En otras palabras, es quien interactúa con algo y se ve afectado por su uso.

Interfaz: es un medio que permite la comunicación e intercambio de información entre dos o más componentes, que pueden ser software, hardware, dispositivos periféricos, personas, o combinaciones de estos. Se trata de un punto de contacto o enlace que facilita la interacción entre elementos distintos.

Responsive: se refiere a la capacidad de un sitio web o aplicación para adaptarse automáticamente al tamaño y la resolución de la pantalla del dispositivo en el que se visualiza. Esto significa que el contenido, el diseño y la funcionalidad se ajustan para ofrecer una experiencia óptima, ya sea en un smartphone, una tableta o una computadora de escritorio.

CSS (Cascading Style Sheets): es un lenguaje de diseño gráfico que se utiliza para describir cómo se ve una página web. Es el código que define el aspecto visual de un documento HTML, como el color del texto, la tipografía, el tamaño, la posición y otros elementos de diseño.

Navegadores: es un software que te permite acceder y visualizar información en la World Wide Web. Es como una puerta de entrada a internet, que interpreta el código HTML de las páginas web y las muestra en pantalla para que puedas navegar y interactuar con su contenido.

Bases de datos: es una colección organizada de información almacenada digitalmente y gestionada por un sistema de gestión de bases de datos (SGBD), que permite almacenar, recuperar, actualizar y gestionar datos. Se utiliza para organizar y acceder a grandes cantidades de datos de manera eficiente.

Atributos: en programación, un atributo es una propiedad o característica que describe un objeto o elemento, que puede tener diferentes valores. En esencia, son las características que definen el estado de un objeto o elemento en un programa.

Entidad: se refiere a una unidad de datos o un objeto que existe en un sistema y tiene propiedades o atributos que lo definen.

Páginas web: es un documento electrónico accesible a través de internet que contiene información organizada y presentada de forma visual.

Protocolos: en el ámbito del software, los protocolos son conjuntos de reglas y normas que definen cómo los sistemas y programas se comunican entre sí. Estos protocolos establecen cómo se formatea la información, se organiza el intercambio de datos y se manejan los errores o situaciones inesperadas.

Software: es el conjunto de los programas de cómputo, procedimientos, reglas, documentación y datos asociados, que forman parte de las operaciones de un sistema de computación.

Blog: Un blog es un sitio web o sección de un sitio web donde se publica contenido regularmente, como artículos, entradas o posts, sobre uno o varios temas. Sirve para compartir información, opiniones, experiencias, o para promocionar productos y servicios.

Industria del Software: La industria del software abarca la creación, distribución y mantenimiento de programas informáticos y aplicaciones. Es un sector económico clave que impulsa la innovación y la transformación digital en diversas áreas, desde la industria manufacturera hasta la atención médica.

Tecnologías modernas: Las tecnologías modernas son aquellas innovaciones en sistemas, dispositivos y recursos que transforman la forma en que vivimos y trabajamos, abarcando áreas como la informática, la inteligencia artificial y las energías renovables.

Javascript: es un lenguaje de programación de scripting que se ejecuta en el navegador web, permitiendo a los desarrolladores crear interacciones y dinamismo en las páginas web. Es fundamental para el desarrollo web moderno, complementando a HTML (estructura) y CSS (diseño). En esencia, JavaScript permite manipular el contenido de la página, añadir efectos interactivos, controlar el flujo de la aplicación y mucho más.

Bootstrap: es un framework de código abierto que simplifica la creación de sitios web y aplicaciones web responsivas, lo que significa que se adaptan a diferentes dispositivos y tamaños de pantalla. Proporciona una serie de componentes de diseño, como botones, menús, tarjetas y más, así como un sistema de rejilla para organizar el contenido, todo ello with clases CSS predefinidas para una fácil integración.

Tailwind: es un framework CSS de bajo nivel, también conocido como "utility-first". Esto significa que, a diferencia de otros frameworks como Bootstrap, no proporciona componentes predefinidos (botones, tablas, etc.) con estilos específicos. En cambio, ofrece una gran cantidad de clases de utilidad individuales que se aplican a los elementos HTML para lograr estilos específicos.

SEO (Search Engine Optimization): significa la optimización para motores de búsqueda. Es un conjunto de estrategias y técnicas que se aplican para que una página web sea fácilmente localizable y aparezca en posiciones relevantes en los resultados de búsqueda de motores como Google, Yahoo o Bing. El objetivo principal del SEO es aumentar la visibilidad de la página web y atraer tráfico orgánico (no de pago) a través de los motores de búsqueda.

APIs (Application Programming Interface): es un conjunto de reglas que permite que diferentes aplicaciones de software se comuniquen e intercambien datos, funcionalidades y características entre sí.

Online: significa "en línea" o "conectado a internet". Implica que algo está disponible o se realiza a través de una red, especialmente internet.

UX/UI: UX (User Experience) se refiere a la experiencia del usuario al interactuar con un producto o servicio, mientras que UI (User Interface) se centra en el diseño visual y la presentación de esa interacción. Ambos son fundamentales para crear productos digitales efectivos y atractivos. UI (User Interface) se refiere a la interfaz de usuario, que es el diseño visual y la disposición de los elementos con los que interactúa un usuario en una aplicación o sitio web. Incluye botones, menús, iconos y otros componentes visuales que facilitan la navegación y la interacción.

Web Performance: se refiere a la velocidad y eficiencia con la que se cargan y funcionan las páginas web. Incluye aspectos como el tiempo de carga, la fluidez de la navegación y la capacidad de respuesta de los elementos interactivos.

Accesibilidad global: es el principio de diseñar y desarrollar productos, servicios y entornos digitales de manera que sean utilizables por todas las personas, independientemente de sus capacidades o discapacidades. Esto incluye garantizar que los sitios web, aplicaciones y tecnologías sean accesibles para personas con discapacidades visuales, auditivas, motoras o cognitivas, así como para aquellos que utilizan diferentes dispositivos o tecnologías de asistencia.

Desarrollo web: es el proceso de crear y mantener sitios web y aplicaciones en línea. Implica la programación, diseño, implementación y gestión de contenido para ofrecer una experiencia interactiva y funcional a los usuarios.

CMS (Content Management System): es un software que permite crear, gestionar y publicar contenido digital en un sitio web sin necesidad de conocimientos técnicos avanzados. Facilita la edición, organización y actualización de información de manera sencilla.

Google: es un motor de búsqueda y una empresa tecnológica que ofrece una amplia gama de servicios en línea, incluyendo búsqueda web, publicidad, correo electrónico (Gmail), almacenamiento en la nube (Google Drive), y más. Es conocido por su algoritmo de búsqueda avanzado y su capacidad para indexar y organizar grandes cantidades de información en internet.

Metodologías agiles: son enfoques de desarrollo de software que se centran en la colaboración, la flexibilidad y la entrega continua. Promueven la adaptabilidad a los cambios y la mejora constante del proceso de desarrollo, priorizando la satisfacción del cliente y la calidad del producto final.

Interactividad: se refiere a la capacidad de un sistema, aplicación o sitio web para permitir que los usuarios interactúen y participen activamente en la experiencia. Esto incluye acciones como hacer clic, desplazarse, enviar formularios, comentar, compartir contenido y recibir retroalimentación en tiempo real.

Pseudocódigo: es una representación informal de un algoritmo que utiliza un lenguaje similar al de programación, pero sin seguir una sintaxis estricta. Se utiliza para planificar y comunicar la lógica de un programa antes de escribir el código real.

Scrum: es un marco de trabajo ágil utilizado para gestionar proyectos de desarrollo de software. Se basa en la colaboración, la adaptabilidad y la entrega incremental de productos, permitiendo a los equipos trabajar de manera eficiente y flexible.

Kanban: es un método de gestión visual utilizado para optimizar el flujo de trabajo y mejorar la eficiencia en proyectos. Se basa en el uso de tarjetas o tableros para representar tareas y su estado, permitiendo a los equipos visualizar el progreso y gestionar las prioridades de manera efectiva.

Testing: es el proceso de evaluar un sistema, software o producto para identificar errores, fallos o áreas de mejora. Se realiza para garantizar que el producto cumpla con los requisitos y funcione correctamente antes de su lanzamiento.

Feedback: es la retroalimentación o respuesta que se recibe de los usuarios o clientes sobre un producto, servicio o experiencia. Se utiliza para evaluar el rendimiento, identificar áreas de mejora y tomar decisiones informadas para optimizar la calidad y satisfacción del usuario.

Administrador: es una persona encargada de gestionar y supervisar un sistema, organización o proyecto. Su función principal es coordinar recursos, tomar decisiones estratégicas y garantizar el buen funcionamiento de las operaciones.

Código fuente: es el conjunto de instrucciones escritas en un lenguaje de programación que define el comportamiento y la funcionalidad de un software. Es el código legible por humanos que se compila o interpreta para crear un programa ejecutable.

Internet: es una red global de computadoras interconectadas que permite la comunicación y el intercambio de información a través de protocolos estandarizados. Es la infraestructura que soporta servicios como la web, el correo electrónico y las redes sociales.

Computadora: es un dispositivo electrónico que procesa datos y realiza cálculos a alta velocidad. Se utiliza para ejecutar programas, almacenar información y realizar tareas diversas, desde navegación por internet hasta edición de documentos.

FTP (File Transfer Protocol): es un protocolo de red utilizado para transferir archivos entre computadoras a través de internet. Permite a los usuarios subir y descargar archivos de servidores remotos de manera eficiente.

Estándares de calidad: son criterios y directrices establecidos para garantizar que un producto, servicio o proceso cumpla con ciertos niveles de excelencia y satisfacción del cliente. Estos estándares ayudan a mantener la consistencia, la seguridad y la eficacia en diversas industrias.

Simplicidad en la Web: se refiere a la práctica de diseñar y desarrollar sitios web de manera clara, intuitiva y fácil de usar. Busca eliminar elementos innecesarios, mejorar la navegación y garantizar que los usuarios puedan encontrar rápidamente la información que necesitan.

Jerarquía visual: es un principio de diseño que organiza los elementos visuales de una página o interfaz de manera que guíen la atención del usuario y resalten la información más importante. Utiliza tamaños, colores, tipografías y disposición para crear un orden lógico y facilitar la comprensión.

Navegación intuitiva: es un diseño de interfaz que permite a los usuarios moverse por un sitio web o aplicación de manera natural y sin esfuerzo. Se basa en la lógica y la familiaridad, facilitando que los usuarios encuentren lo que buscan sin confusión.

Navegabilidad: es la facilidad con la que los usuarios pueden moverse y encontrar información en un sitio web o aplicación. Implica una estructura clara, enlaces intuitivos y una organización lógica del contenido para mejorar la experiencia del usuario.

Consistencia: es la práctica de mantener un diseño y una experiencia de usuario uniformes en todas las páginas de un sitio web o aplicación. Esto incluye el uso coherente de colores, tipografías, botones y elementos visuales para crear una identidad clara y facilitar la navegación.

HTTPS (Hypertext Transfer Protocol Secure): es una versión segura del protocolo HTTP utilizado para la transferencia de datos en la web. Utiliza cifrado para proteger la comunicación entre el navegador del usuario y el servidor, garantizando la confidencialidad e integridad de la información transmitida.

W3C (World Wide Web Consortium): es una organización internacional que desarrolla estándares y pautas para garantizar el crecimiento y la interoperabilidad de la web. Su objetivo es promover el desarrollo de tecnologías web accesibles, eficientes y universales.

Etiquetas: en el contexto de la web, las etiquetas son elementos HTML que se utilizan para estructurar y dar formato al contenido de una página. Estas etiquetas definen cómo se muestra el texto, imágenes, enlaces y otros elementos en un sitio web.

Desarrollo Frontend: es la parte del desarrollo web que se encarga de crear la interfaz de usuario y la experiencia visual de un sitio o aplicación. Implica el uso de tecnologías como HTML, CSS y JavaScript para diseñar y construir lo que los usuarios ven e interactúan directamente.

Desarrollo Backend: es la parte del desarrollo web que se encarga de la lógica del servidor, la base de datos y la gestión de datos. Implica el uso de lenguajes de programación como Python, Ruby, PHP o Java para crear la funcionalidad detrás de las aplicaciones web.

HTTP (Hypertext Transfer Protocol): es el protocolo de comunicación utilizado para la transferencia de datos en la World Wide Web. Permite que los navegadores web soliciten y reciban información de los servidores, facilitando la navegación por internet.

WWW (World Wide Web): es un sistema de información que permite acceder a documentos y recursos interconectados a través de internet. Utiliza hipervínculos para navegar entre páginas web, facilitando la búsqueda y el intercambio de información en línea.

Red LAN (Local Area Network): es una red de computadoras que conecta dispositivos dentro de un área geográfica limitada, como una oficina o un edificio. Permite la comunicación y el intercambio de datos entre computadoras y otros dispositivos en esa área local.

IP (Internet Protocol): es un conjunto de reglas que rige la forma en que los datos se envían y reciben a través de una red. Cada dispositivo conectado a internet tiene una dirección IP única que lo identifica y permite la comunicación entre ellos.
